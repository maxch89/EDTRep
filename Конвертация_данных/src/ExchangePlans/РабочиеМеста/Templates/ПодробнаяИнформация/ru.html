Синхронизация данных с 1С:Конвертация данных редакция 3.0 предназначена для выполнения синхронизации данных между однотипными конфигурациями Конвертация данных редакция 3.0 по правилам обмена. Использование правил обмена позволяет производить гибкую настройку и реализовывать  сложные сценарии синхронизации данных. Создавать и редактировать правила обмена можно в конфигурации 1С: Конвертация данных. 

В какой из двух программ начинать процесс настройки

При настройке синхронизации данных между двумя программами необходимо определить в какой из двух программ начинать процесс настройки. Решение нужно принимать в зависимости от наполненности двух программ данными:

1. Если в обеих программах на момент начала настройки уже имеются данные, то в этом случае в качестве программы для начала настройки синхронизации данных лучше выбрать ту программу, в которой данные заполнены качественнее, более подробно.

При выборе необходимо учитывать, что если часть из имеющихся данных двух программ является общими, то от того, какая программа выбрана в качестве источника, будет зависеть и то, к какому виду будет приведена эта часть общих данных.

Если не все общие данные нужно принять в том виде, в каком они содержатся в программе-источнике, то рекомендуется воспользоваться синхронизацией данных с установкой дополнительных параметров. При первой синхронизации данных необходимо загрузить из другой программы только те данные, которые должны быть загружены в том виде, в котором они хранились в ней. Остальные данные будут синхронизированы позже.

2. Если одна из программ (информационных баз) пустая или практически пустая, то выбор программы для начала процесса настройки упрощается. Стоит лишь учитывать то, что создание настройки синхронизации данных происходит быстрее в менее заполненной программе, т.к. при настройке тратится время на выгрузку данных.
