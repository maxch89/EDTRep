Справочник "Группы правил"

Предназначен для произвольной группировки разных видов правил:

  • Правил обработки данных
  • Правил конвертации объектов
  • Правил конвертации предопределенных данных
  • Алгоритмов
  • Параметров конвертации

Может быть создана единая иерархия групп правил, которая распространяется на все виды правил.

Группа правил указывается для каждого правила в форме элемента.

Список правил с отбором по группам отображается в форме "Настройка правил обмена".
