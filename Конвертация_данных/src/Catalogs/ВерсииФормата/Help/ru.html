Справочник "Версии формата"

Справочник содержит перечень версий формата, которые используются при настройке правил обмена. Данному справочнику подчинено большое количество дополнительных справочников, хранящих содержимое формата.

Для версии должно быть указано Пространство имен: в соответствии с тем, которое указано для схемы XDTO в конфигурациях, поддерживающих данную версию формата.
