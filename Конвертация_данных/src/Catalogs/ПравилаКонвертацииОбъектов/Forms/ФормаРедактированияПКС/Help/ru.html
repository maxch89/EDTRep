Настройка правил конвертации свойств

Свойство конфигурации и Свойство формата: выбираются из списка свойств соответствующего объекта. Если свойств в списке слишком много, рекомендуется набрать первые буквы имени свойства чтобы ограничить список.

При необходимости свойство может быть указано вручную (переключатель Ввод вручную).

Если свойство заполняется алгоритмом, следует установить флаг Используется алгоритм конвертации. В этом случае следует указать только одно из свойств - то которое заполняется (Свойство формата в случае правила выгрузки, и Свойство объекта в случае правила загрузки).

Правило конвертации свойства следует указать, если свойства имеют ссылочный тип, и для этого типа предусмотрено свое правило конвертации объекта. При необходимости (составной тип данных) правило конвертации для  свойства можно определять в алгоритмах правила конвертации объекта.
