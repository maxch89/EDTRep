Сравнение версий формата

Обработка предназначена для сравнения разных версий формата в целях выявления различий. В одном окне можно сравнить до 4-х разных версий. По умолчанию можно указать только две версии, для сравнения большего количества версий следует воспользоваться командой Изменить форму.

Следует выбрать предназначенные для сравнения версии формата и нажать кнопку обновления данных.

Показывать только изменения. При установке данного флага будут показаны только измененные объекты формата.

Показывать требования. При установке данного флага будут показаны требования (постановки задач), в соответствии с которыми внесены изменения. Флаг отображается, если в информационной базе заполнен справочник Требования.

Показывать расхождения с проектными решениями. При установке данного флага будут показаны изменения, которые не соответствуют зарегистрированным проектным решениям (изменение без проектного решения, или проектное решение без изменения в формате). Флаг отображается, если используется функционал Проектирование формата.

Отчет о сравнении

Формирование отчета об изменениях в версии формата в формате html. Доступен только после заполнения дерева с результатом сравнения версий. Поддерживается отчет о сравнении двух версий формата. Отчет доступен, если в информационной базе используется функционал Проектирование формата.
