Краткий состав изменений


Перечень измененных объектов формата.
По гиперссылке Объект можно перейти в раздел Подробный состав изменений, где изменения объекта описаны детально.
Для добавленных и удаленных объектов формата переход к подробным изменениям не предусмотрен, см. Дерево объектов формата или Описание формата.
По гиперссылке Требование можно перейти в раздел Список требований, где описан полный состав изменений в формате для каждого требования. КраткийСоставИзмененийКонец ПодробныйСоставИзмененийНачало

Подробные изменения в объектах формата (с точностью до свойств).
