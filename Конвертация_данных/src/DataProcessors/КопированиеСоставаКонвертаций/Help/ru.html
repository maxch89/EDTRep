Обработка "Копирование состава конвертаций"

Предназначена для распространения правил одной конвертации на другую конвертацию. Используется при создании новой конвертации, которая является потомком существующей. При копировании состава конвертаций не происходит создание новых правил - вместо этого существующие правила включаются в состав указанной конвертации.

Для копирования состава конвертаций необходимо:

  • Выбрать Конвертацию источник - конвертация, которая является источником правил
  • Выбрать Конвертацию получатель - конвертация, которая является получателем правил
  • Указать состав данных:
    • либо "все"
    • либо "выборочно", и настроить перечень правил которые подлежат копированию
