Обработка "Дерево объектов формата"

Обработка предназначена для удобного визуального представления структуры формата.

Основные возможности:

  • Представление формата в иерархическом виде.
  • Поиск ссылок на объект формата среди других объектов формата, переход по найденным ссылкам (флаг Отображать поиск ссылок).
  • Установка закладок на ветвях дерева с последующим переходом между закладками (кнопки на верхней командной панели).
