Обработка "Загрузка обработчиков"

Выполняет загрузку обработчиков из буфера обмена в объекты выбранной конвертации.

Загружаются обработчики для  следующих объектов:

  • Правила обработки данных
  • Правила конвертации объектов
  • Алгоритмы

Если различий в тексте модулей по сравнению с содержимым объектов не обнаружено, объекты конвертации не перезаписываются.

Обновляются только те алгоритмы, которые относятся к существующим объектам конвертации. Если в модуле был создан новый обработчик, он загружен не будет.

Список измененных объектов отображается в окне сообщений.

Определение конвертации, для которой происходит загрузка

Зависит от способа запуска:

При запуске через меню "Сервис"

Предпринимается попытка определить конвертацию по первой  строке из буфера обмена (по шаблону //Конвертация_<Имя конвертации>).

Если это не получается, открывается форма обработки с возможностью указать конвертацию явно

При запуске из формы "Настройка правил обмена"

Конвертация определяется по текущей конвертации
